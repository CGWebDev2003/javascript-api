Class: Toast

Toast(config)

new Toast(config)

Create a new Neptune Toast
Parameters:
Name Type Description
config any Add your configuration parent -> class or id of your target element, when null its document.body icon -> add your icon text -> add your message text style -> primary, accent, information, success, warning, error position -> left-top, left-bottom, right-top, right-bottom

  • neptune.js, line 242
Example
const myToast = new Toast({
 icon: "YOUR ICON",
 text: "Test Toast",
 style: "primary",
 position: "top-right"
});
